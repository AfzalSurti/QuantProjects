yfinance    → connects to Yahoo Finance and downloads real price data
              (Yahoo Finance is free — professional firms pay millions
              for Bloomberg/Refinitiv data. For learning, Yahoo is perfect)

pandas      → handles tables of data (like Excel but in Python)
              Every price dataset = a pandas DataFrame

numpy       → math library — arrays, statistics, calculations
              pandas is built on top of numpy

matplotlib  → draws charts and graphs
